# DSA210 Term Project   - Turkey EV Charging Infrastructure Analysis
#### Spring 2026-2027
#### Sabancı University
#### Student/ID: Melike Yıldırım 32204
## Motivation
After my family switched to electric vehicles, range anxiety -the fear 
of running out of charge - quickly became a real concern in our daily 
lives. This made me wonder: is Turkey's charging infrastructure actually 
ready for the growing number of EV users? 

As electric vehicle adoption 
continues to increase, the availability and distribution of charging 
stations become increasingly important. This project aims to explore 
the current state of EV charging infrastructure in Turkey and examine 
how well it supports the growth of electric vehicles.
