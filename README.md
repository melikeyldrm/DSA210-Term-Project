## Motivation
After my family switched to electric vehicles, range anxiety — the fear 
of running out of charge — quickly became a real concern in our daily 
lives. This made me wonder: is Turkey's charging infrastructure actually 
ready for the growing number of EV users?

As electric vehicle adoption continues to increase, the availability and 
distribution of charging stations become increasingly important. This 
project aims to explore the current state of EV charging infrastructure 
in Turkey and examine how well it supports the growth of electric vehicles.

## Data Sources
The project will use publicly available datasets from multiple sources such as :
- **Open Charge Map** — charging station locations and metadata
- **IEA** — electric vehicle statistics (used to enrich station data with EV adoption trends)
- **TÜİK** — transportation and population statistics (used to normalize infrastructure metrics by region)

## Planned Analysis
The analysis will include the following steps:
- Data collection and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of charging station distribution across Turkey
- Comparison between EV growth and charging infrastructure growth
- Hypothesis testing on infrastructure readiness
- Machine learning models (e.g. clustering regions by infrastructure coverage, regression on charging demand)
